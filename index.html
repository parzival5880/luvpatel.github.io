Featured Project · In Development

IC Mapper

AI debugging co-pilot for hardware datasheets

Engineers debugging circuits spend 20+ minutes manually cross-referencing pin configs, application diagrams, electrical specs, and register maps across 40–60 page IC datasheets. IC Mapper answers those questions in seconds — with source citations, grounding verification, and no hallucinated component values.

Built a 6-stage fully local RAG pipeline: Unstructured.io (hi_res OCR + layout detection) for structure-aware PDF parsing, hybrid dense+sparse retrieval via Qdrant with RRF fusion, cross-encoder reranking, and DeepSeek R1 14B via Ollama for chain-of-thought reasoning. A rule-based grounding agent verifies every numeric citation before the user sees the answer.
